Introduction

Eifel Retransmission Timer suggests an alternative way of RTO estimation in TCP. It uses a RTT based RTO prediction which overcomes the shortcomings of the default TCP RTO estimation. In this project we have implemented this retransmission timer for Constrained Application Protocol(CoAP). CoAP uses a fixed RTO value with BEB in case of retransmissions, after implementing Eifel Retransmission Timer in CoAP, the RTO values will be set based on network state determined from the RTT values.
